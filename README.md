## 本地与部署说明

1. **仓库名**：须为 `lucain55.github.io`（与 GitHub 用户名一致）。
2. **GitHub Pages**：仓库 **Settings → Pages**  
   - Source：**Deploy from a branch**  
   - Branch：**master**，Folder：**/ (root)**  
   - 保存后等待 1～2 分钟即可访问。
3. **更新站点**：改完推送到 `master` 即自动部署。

```bash
git add .
git commit -m "update"
git push origin master
```
